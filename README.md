## 🚀 Cách dùng

1. Mở `index.html` trên trình duyệt bất kỳ

- ⏱️ **Đếm ngược** thời gian thảo luận
- 🗳️ **Bỏ phiếu** trực quan
- 🏆 **Thắng/thua** tự động
- 🇻🇳🇬🇧 **Song ngữ Việt-Anh** dễ chuyển đổi
- ✨ **Hiệu ứng animation** đẹp mắt

## 📁 Cấu trúc

```
masoi-tqd/
├── index.html        # Trang chính
├── css/
│   └── style.css     # Toàn bộ style
├── js/
│   ├── roles.js      # Dữ liệu vai trò
│   ├── game.js       # Logic game
│   ├── ui.js         # Giao diện & dịch ngôn ngữ
│   └── main.js       # Khởi tạo
└── README.md
```

## 🌐 Deploy GitHub Pages

1. Tạo repo mới trên GitHub
2. Push toàn bộ thư mục này lên
3. Vào **Settings** → **Pages** → Source: `main` branch, `/ (root)`
4. Truy cập `https://github.com/duongtool/masoi-tqd/`
